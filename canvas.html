Canvas

Este mensaje aparece si no se soporta el elemento canvas.

Algunos atributos

Existen varios tipos de inputs, como por ejemplo, text.

Otro atributo es placeholder, usado para guiar al ususario con el tipo de información que debe introducir (nombre, apellidos, teléfono...)

Todo lo que yo escriba quedará almacenado en el atributo value, que vale vecío por defecto a no ser que pongamos nosotros un valor.

Si pusiéramos el atributo readonly no se podría cambiar el valor.

La etiqueta <label> se usa para poner un título al input que acompaña. Con el atributo for, con el valor del id del elemento con el que se enlaza (nuestro input). Así, clicando el label activamos el campo asociado.

Radiobuttons

Con el atributo name identificas el grupo al que asocias varios radiobuttons para que estos no se puedan seleccionar al mismo tiempo.
